# Template_Forcepoint_NGFW_20XX
Template Zabbix para monitoramento de firewalls Forcepoint via SNMP, com métricas de CPU, memória, conexões e pacotes. Ajustado para compatibilidade com Zabbix 7.0, facilitando importação em ambientes legados. Ideal para visibilidade e alertas de infraestrutura de segurança.
